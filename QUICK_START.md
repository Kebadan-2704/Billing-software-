The application is now running at: **http://localhost:3000**

### Step 1: Company Selection
- Open your browser and go to http://localhost:3000/company-selection
- You'll see 3 professional company cards
- Click on the company you want to access

### Step 2: Login with Password

### Step 3: Dashboard Overview

Once logged in, you'll see:

#### KPI Cards (Top Section)
1. **Total Projects** - Number of projects for selected period
2. **Amount Received** - Money received from clients
3. **Amount Pending** - Outstanding invoices
4. **Total Amount** - Combined total

#### Financial Statement Table
- **Month-by-month breakdown** for last 3 or 6 months
- **Selector** to switch between 3-month and 6-month views
- **Download button** to export as CSV

#### Navigation Buttons (Top Right)
- 📄 **Create Invoice** - Generate professional invoices
- ⚙️ **Settings** - Manage company GST and details
- 🚪 **Logout** - Return to company selection

## Features Overview

### 📄 Create Professional Invoices
1. Click "Create Invoice" from dashboard
2. View professional invoice template with:
   - Company logo and GST number
   - Client billing information
   - Itemized line items
   - Automatic GST (18%) calculation
   - Professional formatting
3. Options:
   - **Download PDF** - Save invoice locally
   - **Send Email** - Share with client (when integrated)

### ⚙️ Company Settings
1. Click "Settings" button
2. Update company information:
   - GST Registration Number
   - Email and Phone
   - Address details
   - City and State
3. Click "Save Changes"
4. All updates appear on future invoices

### 📊 Financial Statements
1. On Dashboard, select period (3 or 6 months)
2. View detailed breakdown:
   - Each month's projects count
   - Amount received per month
   - Amount pending per month
   - Monthly totals
3. Click "Download" to export as CSV file

## Common Tasks

### Task 1: Download Monthly Statement
1. Go to Dashboard
2. Select "Last 6 Months" or "Last 3 Months"
3. Click "Download" button
4. CSV file downloads to your computer
5. Open in Excel/Google Sheets for analysis

### Task 2: Create and Download Invoice
1. Click "Create Invoice" from Dashboard
2. Review invoice details
3. Click "Download PDF"
4. PDF saves to your computer
5. Share with clients as needed

### Task 3: Update Company GST Number
1. Click "Settings" button
2. Find "GST Registration Number" field
3. Update with new GST number
4. Click "Save Changes"
5. New GST appears on next invoice

### Task 4: Switch Between Companies
1. Click "Logout" button
2. Select different company
3. Enter new company password
4. Access that company's financial data

### Task 5: View Financial Overview
1. Dashboard loads automatically after login
2. See 4 KPI cards with:
   - Number of projects
   - Total money received
   - Total money pending
   - Grand total

## Troubleshooting

### "Incorrect password" message
- Check that you entered the correct demo password
- Password is case-sensitive
- Try one of:
  - `1234` for MD GROUP OF ENTERPRISES
  - `5678` for MD AUTOMIND SOLUTIONS
  - `9012` for MD AUTOMIND ACADEMY

### Invoice not downloading
- Check your browser's download settings
- Ensure pop-ups aren't blocked
- Try a different browser if issues persist
- Verify JavaScript is enabled

### Dashboard shows no data
- Refresh the page (F5)
- Ensure you're logged in (check top of page)
- Try logging out and back in
- Clear browser cache if needed
