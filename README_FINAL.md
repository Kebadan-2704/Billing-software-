## 📞 Troubleshooting

### App won't start?
```bash
# Clear cache and reinstall
rm -r node_modules
npm install
npm run dev
```

### Port 3000 in use?
App automatically uses port 3001 or next available port

### Logo not showing?
- Check: `public/logo.png` exists
- Restart dev server: `npm run dev`
- Clear browser cache: Ctrl+Shift+Delete

### Build errors?
```bash
rm -r .next
npm run build
```
